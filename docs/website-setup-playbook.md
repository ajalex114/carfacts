# ⭐ WEBSITE SETUP & DISTRIBUTION PLAYBOOK
*A reusable blueprint for any website or project*

# 1. Initial Google Setup

## 1.1 Google Search Console (GSC)
**Purpose:** Indexing, performance tracking, backlinks, sitemap submission.

**Steps:**
- Add property: `https://yourdomain.com`
- Verify ownership using **Site Kit** (WordPress plugin)
- Submit sitemap:
  ```
  https://yourdomain.com/sitemap_index.xml
  ```
- Check:
  - Coverage
  - Indexing status
  - Enhancements
  - Mobile usability
- Confirm Google is crawling the site normally

---

## 1.2 Google Analytics (GA4)
**Purpose:** Traffic analytics, user behavior, acquisition channels.

**Steps:**
- Create GA4 property
- Connect via **Site Kit**
- Verify real-time tracking
- Enable:
  - Page views
  - Engagement tracking
  - Scroll depth
  - Outbound clicks

---

## 1.3 Google AdSense
**Purpose:** Monetization.

**Steps:**
- Connect site to AdSense
- Verify site ownership
- Enable auto ads (optional)
- Check publisher ID
- Confirm eligibility for future monetization

---

# 2. Bing Setup

**Purpose:** Additional indexing + backlinks + sitemap submission.


**Verify:**

# 5. RSS Distribution & Backlink Creation

This is the most important part for early SEO.
Submit your RSS feed to multiple directories that create **real, crawlable backlinks**.

## 5.1 Blogarama
- Submit feed
- Public listing created
- Backlink active

## 5.2 AllTop
- Submit feed
- Category placement pending
- Backlink active

## 5.3 Feedspot
- Submit via homepage form
- Awaiting approval
- Backlink will appear after listing goes live

## 5.4 Inoreader Directory
- Add feed
- Public feed page created
- Backlink active

## 5.5 The Old Reader
- Add feed
- Public feed page created
- Backlink active

## 5.6 Follow.it
- Add feed
- Public page created
- Backlink active
- Optional email subscriber system available

# 7. Social Media Distribution (Zero-Cost Plan)

A simple, phased plan — master one platform at a time.

## Phase 1 — Instagram Reels
- 1 reel per day
- Scheduled via Meta Business Suite

## Phase 2 — YouTube Shorts
- Same video as Instagram
- 1 short per day

## Phase 3 — Facebook Page
- Auto-share reels
- 1 post per day

## Phase 4 — Pinterest
- 1 pin per day
- Link to your article

## Phase 5 — LinkedIn (optional)
- 3 posts per week
- Tech/industry content
